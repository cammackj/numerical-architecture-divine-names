# Technical Records

This directory contains public protocols and generated reports supporting the
manuscript. It excludes internal revision plans, editorial correspondence, and
referee working documents.

Protocol files record the rules fixed before their associated tests. Result files
report generated outcomes and point to the complete CSV tables in `data/`.

`PUBLICATION_AUDIT.md` records the final pre-release citation, source-link,
numerical consistency, and PDF checks.

`PSALM_22_INTEGRATION_REVIEW.md` and
`REPDIGIT_LADDER_INTEGRATION_REVIEW.md` record the two findings admitted to
version 1.1.0. `V1_1_REPRODUCIBILITY_VERIFICATION.md` records their regenerated
evidence and the final 95-page PDF audit.

`UPDATE_POLICY.md` defines patch, minor, and major releases and explains how
version-specific and all-version Zenodo DOIs will be used.
